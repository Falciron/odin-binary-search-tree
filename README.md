# odin-binary-search-tree
Concise implementation of a [binary search tree](https://www.theodinproject.com/lessons/ruby-binary-search-trees), written in Ruby, for [The Odin Project](https://www.theodinproject.com/).

Nearing the end of the Ruby track, this project has taught or reinforced for me:
- A novel data structure, not directly implemented in the Ruby standard library
- Recursive adaptation of common data structure methods, including traversal
- Definition of recursive methods that support graceful transmission of blocks
- Remediation techniques for reducing [ABC metrics](https://en.wikipedia.org/wiki/ABC_Software_Metric)

## Pre-requisites for Usage

1. Ruby 3+

## Instructions for Usage

1. Type `ruby main.rb` into your terminal to run the example driver script.

## Rubocop Instructions

1. Type `bundle install` into your terminal to install rubocop.
2. Type `bundle exec rubocop -a` into your terminal to detect issues.
